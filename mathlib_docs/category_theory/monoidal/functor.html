category_theory.lax_monoidal_functor (C : Type u₁) [category_theory.category C] [𝒞 : category_theory.monoidal_category C _inst_1] (D : Type u₂) [category_theory.category D] [𝒟 : category_theory.monoidal_category D _inst_2] :

A lax monoidal functor is a functor F : C ⥤ D between monoidal categories, equipped with morphisms ε : 𝟙 _D ⟶ F.obj (𝟙_ C) and μ X Y : F.obj X ⊗ F.obj Y ⟶ F.obj (X ⊗ Y), satisfying the the appropriate coherences.
