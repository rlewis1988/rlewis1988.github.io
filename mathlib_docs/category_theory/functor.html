category_theory.functor (C : Type u₁) [category_theory.category C] (D : Type u₂) [category_theory.category D] :
Type (max v₁ v₂ u₁ u₂)

functor C D represents a functor between categories C and D.

To apply a functor F to an object use F.obj X, and to a morphism use F.map f.

The axiom map_id_lemma expresses preservation of identities, and map_comp_lemma expresses functoriality.

category_theory.functor.id (C : Type u₁) [𝒞 : category_theory.category C] :
C ⥤ C

𝟭 C is the identity functor on a category C.

F ⋙ G is the composition of a functor F and a functor G (F first, then G).

category_theory.functor.comp_obj {C : Type u₁} [𝒞 : category_theory.category C] {D : Type u₂} [𝒟 : category_theory.category D] {E : Type u₃} [ℰ : category_theory.category E] (F : C ⥤ D) (G : D ⥤ E) (X : C) :
