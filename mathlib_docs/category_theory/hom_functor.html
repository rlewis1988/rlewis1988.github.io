The hom functor, sending (X, Y) to the type X ⟶ Y.

category_theory.functor.hom

Π (C : Type u) [𝒞 : category_theory.category C], Cᵒᵖ × C ⥤ Type v

functor.hom is the hom-pairing, sending (X,Y) to X → Y, contravariant in X and covariant in Y.

category_theory.functor.hom_obj

∀ (C : Type u) [𝒞 : category_theory.category C] (X : Cᵒᵖ × C), (category_theory.functor.hom C).obj X = (opposite.unop (X.fst) ⟶ X.snd)
