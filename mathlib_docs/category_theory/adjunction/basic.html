category_theory.adjunction {C : Type u₁} [𝒞 : category_theory.category C] {D : Type u₂} [𝒟 : category_theory.category D] (F : C ⥤ D) (G : D ⥤ C) :
Type (max u₁ u₂ v₁ v₂)

F ⊣ G represents the data of an adjunction between two functors F : C ⥤ D and G : D ⥤ C. F is the left adjoint and G is the right adjoint.
