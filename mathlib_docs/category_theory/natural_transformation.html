category_theory.nat_trans {C : Type u₁} [𝒞 : category_theory.category C] {D : Type u₂} [𝒟 : category_theory.category D] (F G : C ⥤ D) :
Type (max u₁ v₂)

nat_trans F G represents a natural transformation between functors F and G.

The field app provides the components of the natural transformation.

Naturality is expressed by α.naturality_lemma.

category_theory.nat_trans.id {C : Type u₁} [𝒞 : category_theory.category C] {D : Type u₂} [𝒟 : category_theory.category D] (F : C ⥤ D) :
category_theory.nat_trans F F

nat_trans.id F is the identity natural transformation on a functor F.

(category_theory.nat_trans.id F).app X = 𝟙 (F.obj X)
