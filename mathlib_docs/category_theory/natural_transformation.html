category_theory.nat_trans

Π {C : Type u₁} [𝒞 : category_theory.category C] {D : Type u₂} [𝒟 : category_theory.category D], C ⥤ D → C ⥤ D → Type (max u₁ v₂)

nat_trans F G represents a natural transformation between functors F and G.

The field app provides the components of the natural transformation.

Naturality is expressed by α.naturality_lemma.

category_theory.nat_trans.id

Π {C : Type u₁} [𝒞 : category_theory.category C] {D : Type u₂} [𝒟 : category_theory.category D] (F : C ⥤ D), category_theory.nat_trans F F

nat_trans.id F is the identity natural transformation on a functor F.

category_theory.nat_trans.id_app'

∀ {C : Type u₁} [𝒞 : category_theory.category C] {D : Type u₂} [𝒟 : category_theory.category D] (F : C ⥤ D) (X : C), (category_theory.nat_trans.id F).app X = 𝟙 (F.obj X)

vcomp α β is the vertical compositions of natural transformations.
