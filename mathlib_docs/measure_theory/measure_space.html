Measure projections for a measure space.

For measurable sets this returns the measure assigned by the measure_of field in measure. But we can extend this to _all_ sets, but using the outer measure. This gives us monotonicity and subadditivity for all sets.

measure_theory.all_ae {α : Type u_1} [measure_theory.measure_space α] (p : α → Prop) :
Prop

∀ₘ a:α, p a states that the property p is almost everywhere true in the measure space associated with α. This means that the measure of the complementary of p is 0.

In a probability measure, the measure of p is 1, when p is measurable.
