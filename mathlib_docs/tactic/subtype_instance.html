tactic.interactive.subtype_instance :
tactic unit

builds instances for algebraic substructures

Example: ```lean variables {α : Type*} [monoid α] {s : set α}

class is_submonoid (s : set α) : Prop := (one_mem : (1:α) ∈ s) (mul_mem {a b} : a ∈ s → b ∈ s → a * b ∈ s)

instance subtype.monoid {s : set α} [is_submonoid s] : monoid s := by subtype_instance ````
