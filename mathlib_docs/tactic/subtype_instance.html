tactic.interactive.subtype_instance :
tactic unit

builds instances for algebraic substructures

Example: ```lean variables {α : Type*} [monoid α] {s : set α}

class issubmonoid (s : set α) : Prop := (onemem : (1:α) ∈ s) (mul_mem {a b} : a ∈ s → b ∈ s → a * b ∈ s)

instance subtype.monoid {s : set α} [issubmonoid s] : monoid s := by subtypeinstance ````
