strict_mono

Π {α : Type u} {β : Type v} [_inst_1 : has_lt α] [_inst_2 : has_lt β], (α → β) → Prop

A function f is strictly monotone if a < b implies f a < f b.

strict_mono.lt_iff_lt

∀ {α : Type u} {β : Type v} [_inst_1 : linear_order α] [_inst_2 : preorder β] {f : α → β}, strict_mono f → ∀ {a b : α}, f a < f b ↔ a < b
