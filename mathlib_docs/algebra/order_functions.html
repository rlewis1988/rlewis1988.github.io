def strict_mono {α : Type u} {β : Type v} [has_lt α] [has_lt β] :
(α → β) → Prop

A function f is strictly monotone if a < b implies f a < f b.

Equations
  • strict_mono f = ∀ ⦃a b : α⦄, a < b → f a < f b
