Multivariable polynomials on a type is the left adjoint of the forgetful functor from commutative rings to types.

CommRing.free :
Type u ⥤ CommRing

The free functor Type u ⥤ CommRing.{u} sending a type X to the multivariable (commutative) polynomials with variables x : X.

CommRing.free_obj_coe {α : Type u} :
↥(CommRing.free.obj α) = mv_polynomial α ℤ

Attributes: simp
CommRing.free_map_coe {α β : Type u} {f : α → β} :
⇑(CommRing.free.map f) = mv_polynomial.rename f
