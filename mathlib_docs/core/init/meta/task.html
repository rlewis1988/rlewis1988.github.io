task

Type u → Type u

A task is a promise to produce a value later. They perform the same role as promises in JavaScript.

task.get

Π {α : Type u}, task α → α

task.pure

Π {α : Type u}, α → task α

task.map

Π {α : Type u} {β : Type v}, (α → β) → task α → task β

task.flatten

Π {α : Type u}, task (task α) → task α

task.bind

Π {α : Type u} {β : Type v}, task α → (α → task β) → task β

task.delay

Π {α : Type u}, (unit → α) → task α
