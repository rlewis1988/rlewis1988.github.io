well_founded_tactics :
Type

Argument for usingwellfounded

The tactic rel_tac has to synthesize an element of type (haswellfounded A). The two arguments are: a local representing the function being defined by well founded recursion, and a list of recursive equations. The equations can be used to decide which well founded relation should be used.

The tactic dec_tac has to synthesize decreasing proofs.
