name

Type

Reflect a C++ name object. The VM replaces it with the C++ implementation.

auto_param

Sort u → name → Sort u

Gadget for automatic parameter support. This is similar to the optparam gadget, but it uses the tactic declaration names tacname to synthesize the argument. Like opt_param, this gadget only affects elaboration. For example, the tactic will not be invoked during type class resolution.

auto_param_eq

∀ (α : Sort u) (n : name), auto_param α n = α
