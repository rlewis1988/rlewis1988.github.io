tactic.is_type_app_of (e : expr) (I_name : name) :
tactic bool

Return tt iff e's type is of the form (I_name ...)

tactic.mk_brec_on_rec_value (F_name : name) (i : ℕ) :
tactic expr

Construct brecon "recursive value". Fname is the name of the brecon "dictionary". Type of the Fname hypothesis should be of the form (below (C ...)) where C is a constructor. The result is the "recursive value" for the (i+1)-th recursive value of the constructor C.

tactic.mk_constructor_arg_names (c p : name) (i : ℕ) :
tactic (list name × ℕ)

Return a list of names of the form [p.i, ..., p.{i+n}] where n is the number of fields of the constructor c

tactic.mk_constructors_arg_names (I p : name) :
tactic (list (list name))

Given an inductive datatype I with k constructors and where constructor i has ni fields, return the list [[p.1, ..., p.n1], [p.{n1 + 1}, ..., p.{n1 + n2}], ..., [..., p.{n1 + ... + n_k}]]
