copy_decl_updating_type (replacements : name_map name) (src_decl_name new_decl_name : name) :
tactic unit

Given a set of constant renamings replacements and a declaration name src_decl_name, create a new declaration called new_decl_name s.t. its type is the type of src_decl_name after applying the given constant replacement.

Remark: the new type must be definitionally equal to the type of src_decl_name.

Example: Assume the environment contains def f : nat -> nat := ... def g : nat -> nat := f lemma f_lemma : forall a, f a > 0 := ...

Moreover, assume we have a mapping M containing f ->g Then, the command run_command copy_decl_updating_type M f_lemmag_lemma creates the declaration lemma g_lemma : forall a, g a > 0 := ...
