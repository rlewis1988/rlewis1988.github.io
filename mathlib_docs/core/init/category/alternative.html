has_orelse (f : Type u → Type v) :
Type (max (u+1) v)

alternative (f : Type u → Type v) :
Type (max (u+1) v)

failure {f : Type u → Type v} [alternative f] {α : Type u} :
f α

guard {f : Type → Type v} [alternative f] (p : Prop) [decidable p] :
f unit

If the condition p is decided to be false, then fail, otherwise, return unit.

assert {f : Type → Type v} [alternative f] (p : Prop) [decidable p] :
f (inhabited p)

guardb {f : Type → Type v} [alternative f] (a : bool) :
f unit

optional {f : Type u → Type v} [alternative f] {α : Type u} (x : f α) :
f (option α)
