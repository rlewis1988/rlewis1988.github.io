set.countable {α : Type u} (s : set α) :
Prop

Countable sets

A set is countable if there exists an encoding of the set into the natural numbers. An encoding is an injection with a partial inverse, which can be viewed as a constructive analogue of countability. (For the most part, theorems about countable will be classical and encodable will be constructive.)

set.countable_iff_exists_injective {α : Type u} {s : set α} :
set.countable s ↔ ∃ (f : ↥s → ℕ), function.injective f

set.countable_iff_exists_inj_on {α : Type u} {s : set α} :
set.countable s ↔ ∃ (f : α → ℕ), set.inj_on f s

set.countable_iff_exists_surjective {α : Type u} [ne : inhabited α] {s : set α} :
set.countable s ↔ ∃ (f : ℕ → α), s ⊆ set.range f

set.countable_iff_exists_surjective_to_subtype {α : Type u} {s : set α} (hs : s ≠ ∅) :
set.countable s ↔ ∃ (f : ℕ → ↥s), function.surjective f

A non-empty set is countable iff there exists a surjection from the natural numbers onto the subtype induced by the set.
