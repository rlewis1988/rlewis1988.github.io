holor

Basic properties of holors

Holors are indexed collections of tensor coefficients. Confusingly, they are often called tensors in physics and in the neural network community.

A holor is simply a multidimensional array of values. The size of a holor is specified by a list ℕ, whose length is called the dimension of the holor.

The tensor product of x₁ : holor α ds₁ and x₂ : holor α ds₂ is the holor given by (x₁ ⊗ x₂) (i₁ ++ i₂) = x₁ i₁ * x₂ i₂. A holor is "of rank at most 1" if it is a tensor product of one-dimensional holors. The CP rank of a holor x is the smallest N such that x is the sum of N holors of rank at most 1.

Based on the tensor library found in https://www.isa-afp.org/entries/Deep_Learning.html

holor_index (ds : list ℕ) :
Type

holor_index ds is the type of valid index tuples to identify an entry of a holor of dimensions ds

holor_index.take {ds₂ ds₁ : list ℕ} (a : holor_index (ds₁ ++ ds₂)) :
holor_index ds₁

holor_index.drop {ds₂ ds₁ : list ℕ} (a : holor_index (ds₁ ++ ds₂)) :
holor_index ds₂

holor_index.cast_type {ds₁ ds₂ : list ℕ} (is : list ℕ) (eq : ds₁ = ds₂) (h : list.forall₂ has_lt.lt is ds₁) :
(cast _ ⟨is, h⟩).val = is

holor_index.assoc_right {ds₁ ds₂ ds₃ : list ℕ} (a : holor_index (ds₁ ++ ds₂ ++ ds₃)) :
holor_index (ds₁ ++ (ds₂ ++ ds₃))

holor_index.assoc_left {ds₁ ds₂ ds₃ : list ℕ} (a : holor_index (ds₁ ++ (ds₂ ++ ds₃))) :
holor_index (ds₁ ++ ds₂ ++ ds₃)

holor_index.take_take {ds₁ ds₂ ds₃ : list ℕ} (t : holor_index (ds₁ ++ ds₂ ++ ds₃)) :
holor_index.take (holor_index.assoc_right t) = holor_index.take (holor_index.take t)

holor_index.drop_take {ds₁ ds₂ ds₃ : list ℕ} (t : holor_index (ds₁ ++ ds₂ ++ ds₃)) :
holor_index.take (holor_index.drop (holor_index.assoc_right t)) = holor_index.drop (holor_index.take t)

holor_index.drop_drop {ds₁ ds₂ ds₃ : list ℕ} (t : holor_index (ds₁ ++ ds₂ ++ ds₃)) :
holor_index.drop (holor_index.drop (holor_index.assoc_right t)) = holor_index.drop t

holor.slice_eq {α : Type} {d : ℕ} {ds : list ℕ} (x y : holor α (d :: ds)) (h : holor.slice x = holor.slice y) :
x = y

Two holors are equal if all their slices are equal.

holor.slice_unit_vec_mul {α : Type} {d : ℕ} {ds : list ℕ} [ring α] {i j : ℕ} (hid : i < d) (x : holor α ds) :
holor.slice (holor.mul (holor.unit_vec d j) x) i hid = ite (i = j) x 0

holor.slice_add {α : Type} {d : ℕ} {ds : list ℕ} [has_add α] (i : ℕ) (hid : i < d) (x y : holor α (d :: ds)) :
holor.slice x i hid + holor.slice y i hid = holor.slice (x + y) i hid

holor.slice_zero {α : Type} {d : ℕ} {ds : list ℕ} [has_zero α] (i : ℕ) (hid : i < d) :
holor.slice 0 i hid = 0

holor.slice_sum {α : Type} {d : ℕ} {ds : list ℕ} [add_comm_monoid α] {β : Type} (i : ℕ) (hid : i < d) (s : finset β) (f : β → holor α (d :: ds)) :
finset.sum s (λ (x : β), holor.slice (f x) i hid) = holor.slice (finset.sum s f) i hid

holor.sum_unit_vec_mul_slice {α : Type} {d : ℕ} {ds : list ℕ} [ring α] (x : holor α (d :: ds)) :
finset.sum (finset.attach (finset.range d)) (λ (i : {x // x ∈ finset.range d}), holor.mul (holor.unit_vec d (i.val)) (holor.slice x (i.val) _)) = x

The original holor can be recovered from its slices by multiplying with unit vectors and summing up.

holor.cprank_max1 {α : Type} [has_mul α] {ds : list ℕ} (a : holor α ds) :
Prop

cprank_max1 x means x has CP rank at most 1, that is, it is the tensor product of 1-dimensional holors.

holor.cprank_max {α : Type} [has_mul α] [add_monoid α] (a : ℕ) {ds : list ℕ} (a_1 : holor α ds) :
Prop

cprank_max N x means x has CP rank at most N, that is, it can be written as the sum of N holors of rank at most 1.

holor.cprank_max_nil {α : Type} [monoid α] [add_monoid α] (x : holor α list.nil) :
holor.cprank_max 1 x

holor.cprank_max_1 {α : Type} {ds : list ℕ} [monoid α] [add_monoid α] {x : holor α ds} (h : holor.cprank_max1 x) :
holor.cprank_max 1 x

holor.cprank_max_add {α : Type} {ds : list ℕ} [monoid α] [add_monoid α] {m n : ℕ} {x y : holor α ds} (a : holor.cprank_max m x) (a_1 : holor.cprank_max n y) :
holor.cprank_max (m + n) (x + y)

holor.cprank_max_mul {α : Type} {d : ℕ} {ds : list ℕ} [ring α] (n : ℕ) (x : holor α [d]) (y : holor α ds) (a : holor.cprank_max n y) :
holor.cprank_max n (holor.mul x y)

holor.cprank_max_sum {α : Type} {ds : list ℕ} [ring α] {β : Type u_1} {n : ℕ} (s : finset β) (f : β → holor α ds) (a : ∀ (x : β), x ∈ s → holor.cprank_max n (f x)) :
holor.cprank_max (finset.card s * n) (finset.sum s f)

holor.cprank_max_upper_bound {α : Type} [ring α] {ds : list ℕ} (x : holor α ds) :
holor.cprank_max (list.prod ds) x

The CP rank of a holor x: the smallest N such that x can be written as the sum of N holors of rank at most 1.
