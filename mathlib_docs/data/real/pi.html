real.sqrt_two_add_series

ℝ → ℕ → ℝ

the series sqrt_two_add_series x n is sqrt(2 + sqrt(2 + ... )) with n square roots, starting with x. We define it here because cos (pi / 2 ^ (n+1)) = sqrt_two_add_series 0 n / 2

Attributes: simp

real.sqrt_two_add_series_zero

∀ (x : ℝ), real.sqrt_two_add_series x 0 = x

real.sqrt_two_add_series_one

real.sqrt_two_add_series 0 1 = real.sqrt 2

real.sqrt_two_add_series_two

real.sqrt_two_add_series 0 2 = real.sqrt (2 + real.sqrt 2)

real.sqrt_two_add_series_zero_nonneg

∀ (n : ℕ), real.sqrt_two_add_series 0 n ≥ 0

real.sqrt_two_add_series_nonneg

∀ {x : ℝ}, 0 ≤ x → ∀ (n : ℕ), real.sqrt_two_add_series x n ≥ 0

real.sqrt_two_add_series_lt_two

∀ (n : ℕ), real.sqrt_two_add_series 0 n < 2

real.sqrt_two_add_series_succ

∀ (x : ℝ) (n : ℕ), real.sqrt_two_add_series x (n + 1) = real.sqrt_two_add_series (real.sqrt (2 + x)) n

real.sqrt_two_add_series_monotone_left

∀ {x y : ℝ}, x ≤ y → ∀ (n : ℕ), real.sqrt_two_add_series x n ≤ real.sqrt_two_add_series y n

real.sqrt_two_add_series_step_up

∀ {a b n : ℕ} {z : ℝ} (c d : ℕ), real.sqrt_two_add_series (↑c / ↑d) n ≤ z → b ≠ 0 → d ≠ 0 → (2 * b + a) * d ^ 2 ≤ c ^ 2 * b → real.sqrt_two_add_series (↑a / ↑b) (n + 1) ≤ z

real.sqrt_two_add_series_step_down

∀ {c d n : ℕ} {z : ℝ} (a b : ℕ), z ≤ real.sqrt_two_add_series (↑a / ↑b) n → b ≠ 0 → d ≠ 0 → a ^ 2 * d ≤ (2 * d + c) * b ^ 2 → z ≤ real.sqrt_two_add_series (↑c / ↑d) (n + 1)
