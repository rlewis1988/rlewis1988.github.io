Floor Function for Rational Numbers

Summary

We define the floor function and the floor_ring instance on ℚ.

Tags

rat, rationals, ℚ, floor

rat.floor (a : ℚ) :
ℤ

floor q is the largest integer z such that z ≤ q

rat.le_floor {z : ℤ} {r : ℚ} :
z ≤ rat.floor r ↔ ↑z ≤ r

rat.floor_ring :
floor_ring ℚ

Attributes: instance
rat.floor_def {q : ℚ} :
⌊q⌋ = q.num / ↑(q.denom)
