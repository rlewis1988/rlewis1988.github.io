zsqrtd

ℤ → Type

The ring of integers adjoined with a square root of d. These have the form a + b √d where a b : ℤ. The components are called re and im by analogy to the negative d case, but of course both parts are real here since d is nonnegative.

zsqrtd.sq_le

ℕ → ℕ → ℕ → ℕ → Prop

Read sq_le a c b d as a √c ≤ b √d

∀ {c d x y z w : ℕ}, z ≤ x → y ≤ w → zsqrtd.sq_le x c y d → zsqrtd.sq_le z c w d

zsqrtd.sq_le_add_mixed

∀ {c d x y z w : ℕ}, zsqrtd.sq_le x c y d → zsqrtd.sq_le z c w d → c * (x * z) ≤ d * (y * w)

zsqrtd.sq_le_add

∀ {c d x y z w : ℕ}, zsqrtd.sq_le x c y d → zsqrtd.sq_le z c w d → zsqrtd.sq_le (x + z) c (y + w) d

zsqrtd.sq_le_cancel

∀ {c d x y z w : ℕ}, zsqrtd.sq_le y d x c → zsqrtd.sq_le (x + z) c (y + w) d → zsqrtd.sq_le z c w d

zsqrtd.sq_le_smul

∀ {c d x y : ℕ} (n : ℕ), zsqrtd.sq_le x c y d → zsqrtd.sq_le (n * x) c (n * y) d

zsqrtd.sq_le_mul

∀ {d x y z w : ℕ}, (zsqrtd.sq_le x 1 y d → zsqrtd.sq_le z 1 w d → zsqrtd.sq_le (x * w + y * z) d (x * z + d * y * w) 1) ∧ (zsqrtd.sq_le x 1 y d → zsqrtd.sq_le w d z 1 → zsqrtd.sq_le (x * z + d * y * w) 1 (x * w + y * z) d) ∧ (zsqrtd.sq_le y d x 1 → zsqrtd.sq_le z 1 w d → zsqrtd.sq_le (x * z + d * y * w) 1 (x * w + y * z) d) ∧ (zsqrtd.sq_le y d x 1 → zsqrtd.sq_le w d z 1 → zsqrtd.sq_le (x * w + y * z) d (x * z + d * y * w) 1)

zsqrtd.nonsquare

ℕ → Prop

A nonsquare is a natural number that is not equal to the square of an integer. This is implemented as a typeclass because it's a necessary condition for much of the Pell equation theory.
