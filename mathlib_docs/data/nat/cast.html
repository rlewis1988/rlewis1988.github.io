def nat.cast {α : Type u_1} [has_zero α] [has_one α] [has_add α] :
ℕ → α

Canonical homomorphism from ℕ to a type α with 0, 1 and +.

Equations
  • nat.cast (n + 1) = nat.cast n + 1
  • nat.cast 0 = 0
