topological_monoid (α : Type u) [topological_space α] [monoid α] :
Prop

A topological monoid is a monoid in which the multiplication is continuous as a function α × α → α.

topological_add_monoid (α : Type u) [topological_space α] [add_monoid α] :
Prop

A topological (additive) monoid is a monoid in which the addition is continuous as a function α × α → α.
