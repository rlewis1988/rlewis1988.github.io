Dense embeddings

This file defines three properties of functions:

dense_range f means f has dense image; dense_inducing i means i is also inducing; dense_embedding e means e is also an embedding.

The main theorem continuous_extend gives a criterion for a function f : X → Z to a regular (T₃) space Z to extend along a dense embedding i : X → Y to a continuous function g : Y → Z. Actually i only has to be dense_inducing (not necessarily injective).

dense_range

Π {α : Type u_1} {β : Type u_2} [_inst_1 : topological_space β], (α → β) → Prop

f : α → β has dense range if its range (image) is a dense subset of β.

Π {α : Type u_1} {β : Type u_2} [_inst_1 : topological_space α] [_inst_2 : topological_space β], (α → β) → Prop

i : α → β is "dense inducing" if it has dense range and the topology on α is the one induced by i from the topology on β.

dense_inducing.extend

Π {α : Type u_1} {β : Type u_2} {γ : Type u_3} [_inst_1 : topological_space α] [_inst_2 : topological_space β] {i : α → β} [_inst_4 : topological_space γ], dense_inducing i → (α → γ) → β → γ

If i : α → β is a dense inducing, then any function f : α → γ "extends" to a function g = extend di f : β → γ. If γ is Hausdorff and f has a continuous extension, then g is the unique such extension. In general, g might not be continuous or even extend f.

dense_inducing.extend_eq

∀ {α : Type u_1} {β : Type u_2} {γ : Type u_3} [_inst_1 : topological_space α] [_inst_2 : topological_space β] {i : α → β} (di : dense_inducing i) [_inst_4 : topological_space γ] [_inst_5 : t2_space γ] {b : β} {c : γ} {f : α → γ}, filter.map f (filter.comap i (nhds b)) ≤ nhds c → dense_inducing.extend di f b = c

∀ {α : Type u_1} {β : Type u_2} {γ : Type u_3} [_inst_1 : topological_space α] [_inst_2 : topological_space β] {i : α → β} [_inst_4 : topological_space γ] [_inst_5 : regular_space γ] {f : α → γ} (di : dense_inducing i), (∀ (b : β), ∃ (c : γ), filter.tendsto f (filter.comap i (nhds b)) (nhds c)) → continuous (dense_inducing.extend di f)

dense_inducing.mk'

∀ {α : Type u_1} {β : Type u_2} [_inst_1 : topological_space α] [_inst_2 : topological_space β] (i : α → β), continuous i → (∀ (x : β), x ∈ closure (set.range i)) → (∀ (a : α) (s : set α), s ∈ nhds a → (∃ (t : set β) (H : t ∈ nhds (i a)), ∀ (b : α), i b ∈ t → b ∈ s)) → dense_inducing i

dense_embedding

Π {α : Type u_1} {β : Type u_2} [_inst_1 : topological_space α] [_inst_2 : topological_space β], (α → β) → Prop

A dense embedding is an embedding with dense image.

dense_embedding.mk'

∀ {α : Type u_1} {β : Type u_2} [_inst_1 : topological_space α] [_inst_2 : topological_space β] (e : α → β), continuous e → (∀ (x : β), x ∈ closure (set.range e)) → function.injective e → (∀ (a : α) (s : set α), s ∈ nhds a → (∃ (t : set β) (H : t ∈ nhds (e a)), ∀ (b : α), e b ∈ t → b ∈ s)) → dense_embedding e

∀ {α : Type u_1} {β : Type u_2} [_inst_1 : topological_space α] [_inst_2 : topological_space β] {e : α → β}, dense_embedding e → ∀ {x y : α}, e x = e y ↔ x = y
