separation_rel (α : Type u) [u : uniform_space α] :
set (α × α)

The separation relation is the intersection of all entourages. Two points which are related by the separation relation are "indistinguishable" according to the uniform structure.

separated_equiv {α : Type u} [uniform_space α] :
equivalence (λ (x y : α), (x, y) ∈ separation_rel α)

separated (α : Type u) [uniform_space α] :
Prop

separated_def {α : Type u} [uniform_space α] :
separated α ↔ ∀ (x y : α), (∀ (r : set (α × α)), r ∈ uniformity α → (x, y) ∈ r) → x = y

separated_def' {α : Type u} [uniform_space α] :
separated α ↔ ∀ (x y : α), x ≠ y → (∃ (r : set (α × α)) (H : r ∈ uniformity α), (x, y) ∉ r)
