filter.has_countable_basis

Π {α : Type u}, filter α → Prop

A filter has a countable basis iff it is generated by a countable collection of subsets of α. (A filter is a generated by a collection of sets iff it is the infimum of the principal filters.)

Note: we do not require the collection to be closed under finite intersections.

filter.has_countable_basis_of_seq

∀ {α : Type u} (f : filter α) (x : ℕ → set α), (f = ⨅ (i : ℕ), filter.principal (x i)) → filter.has_countable_basis f

filter.seq_of_has_countable_basis

∀ {α : Type u} (f : filter α), filter.has_countable_basis f → (∃ (x : ℕ → set α), f = ⨅ (i : ℕ), filter.principal (x i))

filter.has_countable_basis_iff_seq

∀ {α : Type u} (f : filter α), filter.has_countable_basis f ↔ ∃ (x : ℕ → set α), f = ⨅ (i : ℕ), filter.principal (x i)

Different characterization of countable basis. A filter has a countable basis iff it is generated by a sequence of sets.

filter.mono_seq_of_has_countable_basis

∀ {α : Type u} (f : filter α), filter.has_countable_basis f → (∃ (x : ℕ → set α), (∀ (i j : ℕ), i ≤ j → x j ⊆ x i) ∧ f = ⨅ (i : ℕ), filter.principal (x i))

filter.has_countable_basis_iff_mono_seq

∀ {α : Type u} (f : filter α), filter.has_countable_basis f ↔ ∃ (x : ℕ → set α), (∀ (i j : ℕ), i ≤ j → x j ⊆ x i) ∧ f = ⨅ (i : ℕ), filter.principal (x i)

Different characterization of countable basis. A filter has a countable basis iff it is generated by a monotonically decreasing sequence of sets.

topological_space.is_topological_basis

Π {α : Type u} [t : topological_space α], set (set α) → Prop

A topological basis is one that satisfies the necessary conditions so that it suffices to take unions of the basis sets to get a topology (without taking finite intersections as well).

topological_space.is_topological_basis_of_subbasis

∀ {α : Type u} [t : topological_space α] {s : set (set α)}, t = topological_space.generate_from s → topological_space.is_topological_basis ((λ (f : set (set α)), ⋂₀ f) '' {f : set (set α) | set.finite f ∧ f ⊆ s ∧ ⋂₀ f ≠ ∅})

topological_space.is_topological_basis_of_open_of_nhds

∀ {α : Type u} [t : topological_space α] {s : set (set α)}, (∀ (u : set α), u ∈ s → is_open u) → (∀ (a : α) (u : set α), a ∈ u → is_open u → (∃ (v : set α) (H : v ∈ s), a ∈ v ∧ v ⊆ u)) → topological_space.is_topological_basis s

topological_space.mem_nhds_of_is_topological_basis

∀ {α : Type u} [t : topological_space α] {a : α} {s : set α} {b : set (set α)}, topological_space.is_topological_basis b → (s ∈ nhds a ↔ ∃ (t : set α) (H : t ∈ b), a ∈ t ∧ t ⊆ s)

topological_space.is_open_of_is_topological_basis

∀ {α : Type u} [t : topological_space α] {s : set α} {b : set (set α)}, topological_space.is_topological_basis b → s ∈ b → is_open s

topological_space.mem_basis_subset_of_mem_open

∀ {α : Type u} [t : topological_space α] {b : set (set α)}, topological_space.is_topological_basis b → ∀ {a : α} {u : set α}, a ∈ u → is_open u → (∃ (v : set α) (H : v ∈ b), a ∈ v ∧ v ⊆ u)

topological_space.sUnion_basis_of_is_open

∀ {α : Type u} [t : topological_space α] {B : set (set α)}, topological_space.is_topological_basis B → ∀ {u : set α}, is_open u → (∃ (S : set (set α)) (H : S ⊆ B), u = ⋃₀ S)

topological_space.Union_basis_of_is_open

∀ {α : Type u} [t : topological_space α] {B : set (set α)}, topological_space.is_topological_basis B → ∀ {u : set α}, is_open u → (∃ (β : Type u) (f : β → set α), (u = ⋃ (i : β), f i) ∧ ∀ (i : β), f i ∈ B)

topological_space.separable_space

Π (α : Type u) [t : topological_space α], Prop

A separable space is one with a countable dense subset.

topological_space.first_countable_topology

Π (α : Type u) [t : topological_space α], Prop

A first-countable space is one in which every point has a countable neighborhood basis.

topological_space.second_countable_topology

Π (α : Type u) [t : topological_space α], Prop

A second-countable space is one with a countable basis.
