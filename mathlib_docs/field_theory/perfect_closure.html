A perfect field is a field of characteristic p that has p-th root.

perfect_closure.r (α : Type u) [monoid α] (p : ℕ) (a a_1 : ℕ × α) :
Prop

perfect_closure.r_iff (α : Type u) [monoid α] (p : ℕ) (a a_1 : ℕ × α) :
perfect_closure.r α p a a_1 ↔ ∃ (n : ℕ) (x : α), a = (n, x) ∧ a_1 = (n + 1, frobenius α p x)
