Algebraic elements and algebraic extensions

An element of an R-algebra is algebraic over R if it is the root of a nonzero polynomial. An R-algebra is algebraic over R if and only if all its elements are algebraic over R. The main result in this file proves transitivity of algebraicity: a tower of algebraic field extensions is algebraic.

A subalgebra is algebraic if all its elements are algebraic.

An algebra is algebraic if and only if it is algebraic as a subalgebra.

is_integral.is_algebraic

∀ (R : Type u) {A : Type v} [_inst_1 : nonzero_comm_ring R] [_inst_2 : comm_ring A] [_inst_3 : algebra R A] {x : A}, is_integral R x → is_algebraic R x

An integral element of an algebra is algebraic.

is_algebraic_iff_is_integral

∀ (K : Type u) {A : Type v} [_inst_1 : discrete_field K] [_inst_2 : comm_ring A] [_inst_3 : algebra K A] {x : A}, is_algebraic K x ↔ is_integral K x

An element of an algebra over a field is algebraic if and only if it is integral.

algebra.is_algebraic_trans

∀ {K : Type u_1} {L : Type u_2} {A : Type u_3} [_inst_1 : discrete_field K] [_inst_2 : discrete_field L] [_inst_3 : comm_ring A] [_inst_4 : algebra K L] [_inst_5 : algebra L A], algebra.is_algebraic K L → algebra.is_algebraic L A → algebra.is_algebraic K (algebra.comap K L A)

If L is an algebraic field extension of K and A is an algebraic algebra over L, then A is algebraic over K.
